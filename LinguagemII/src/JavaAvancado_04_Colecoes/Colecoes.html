Collections em Java

As collections são uma parte importante da biblioteca padrão do Java e permitem trabalhar com conjuntos de elementos de forma eficiente e fácil.

Listas

Uma lista em Java é uma coleção ordenada de elementos. Para criar uma lista, você pode usar a classe ArrayList:

		List<String> minhaLista = new ArrayList<>();
		minhaLista.add("elemento 1");
		minhaLista.add("elemento 2");
		minhaLista.add("elemento 3");

Este código cria uma lista de strings e adiciona três elementos à lista. Para acessar um elemento da lista, você pode usar o método get():

		String elemento2 = minhaLista.get(1);
		System.out.println(elemento2);

Este código retorna o segundo elemento da lista ("elemento 2") e o imprime na saída padrão.

Conjuntos

Um conjunto em Java é uma coleção não ordenada de elementos únicos. Para criar um conjunto, você pode usar a classe HashSet:

		Set<String> meuSet = new HashSet<>();
		meuSet.add("elemento 1");
		meuSet.add("elemento 2");
		meuSet.add("elemento 3");

Este código cria um conjunto de strings e adiciona três elementos ao conjunto. Como o conjunto não é ordenado, você não pode acessar elementos do conjunto pelo índice. Para verificar se um elemento está presente no conjunto, você pode usar o método contains():

		boolean contemElemento2 = meuSet.contains("elemento 2");
		System.out.println(contemElemento2);

Este código verifica se o conjunto contém o elemento "elemento 2" e imprime o resultado na saída padrão.

Mapas

Um mapa em Java é uma coleção de pares chave/valor. Para criar um mapa, você pode usar a classe HashMap:

		Map<String, Integer> meuMap = new HashMap<>();
		meuMap.put("chave 1", 10);
		meuMap.put("chave 2", 20);
		meuMap.put("chave 3", 30);

Agora, o mapa contém três pares de chave/valor. Para acessar o valor associado a uma chave, use o método get():

		int valorChave2 = meuMap.get("chave 2");

Este código retorna o valor associado à chave "chave 2".

Iterando sobre uma coleção

Uma das tarefas mais comuns ao trabalhar com coleções é iterar sobre seus elementos. Em Java, você pode iterar sobre uma coleção usando um loop for-each ou um iterator.

Loop for-each

O loop for-each é uma maneira conveniente de iterar sobre os elementos de uma coleção:

		for (String elemento : minhaLista) {
			System.out.println(elemento);
		}

Este código itera sobre todos os elementos da lista e os imprime na saída padrão.

Iterator

Outra maneira de iterar sobre uma coleção é usar um iterator:

		Iterator<String> iterador = minhaLista.iterator();
		while (iterador.hasNext()) {
			String elemento = iterador.next();
			System.out.println(elemento);
		}

Este código também itera sobre todos os elementos da lista e os imprime na saída padrão.

Conclusão

As collections são uma parte essencial do Java e oferecem maneiras convenientes de trabalhar com conjuntos de elementos. Existem vários tipos de coleções, incluindo listas, conjuntos e mapas, e várias maneiras de iterar sobre eles. É importante estar familiarizado com as collections em Java para aproveitar ao máximo a linguagem.
